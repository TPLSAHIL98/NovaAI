# NovaAI

NovaAI is an AI-powered companion mod for Minecraft Fabric 1.21.1.

## ✨ Features

- 👤 Steve-like AI companion
- 💬 Natural AI conversations
- 🚶 Follows the player
- 👀 Looks at the player while talking
- 🧠 Short-term memory
- ⚡ Fast API responses

## 🛠️ Tech Stack

- Minecraft 1.21.1
- Fabric Loader
- Java 21
- Gradle
- Yarn Mappings
